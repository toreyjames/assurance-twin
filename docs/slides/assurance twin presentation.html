Deployment Options for Large Clients

Option 1: SaaS Multi-Tenant (Deloitte-Hosted)

Best for: Multiple clients, managed by Deloitte

  • Infrastructure: Azure Government Cloud or AWS GovCloud
  • Tenant Isolation: Separate database schemas per client
  • Auth: SSO via Azure AD, Okta, Auth0
  • Compliance: SOC 2, ISO 27001, FedRAMP (for regulated)
  • Updates: Centralized - Deloitte manages
  • Cost: ~$2,000/month base (scales with tenants)

Pros: Faster time-to-value, no client IT overhead
Cons: Some clients refuse cloud (data sovereignty)

Option 2: On-Premise / Private Cloud (Client-Hosted)

Best for: Highly regulated industries (Oil & Gas, Utilities, Defense)

  • Deployment: Docker/Kubernetes in client environment
  • Data Control: Client owns all data, never leaves network
  • Air-Gap: Can run fully offline
  • Auth: Integrates with client AD/LDAP
  • Backup/DR: Client's responsibility (or Deloitte manages as service)
  • Updates: Client-controlled deployment schedule

Pros: Meets strictest data sovereignty, air-gap compatible
Cons: Requires client IT resources, slower to update
